Project for Software I

Requirements: Create a Inventory System Application to replace an outgrown spreadsheet system. Allows users to add, delete, or modify any inventory inserted into the application. Must have a working search bar, allows for inventory to be preserved and saved, and must have a safeguard for inventory associated with another differents parts, meaning you can not delete inventory if it is tied to another piece of inventory.

Completed using Java, JavaFX
IDE: IntelliJ IDEA Community Edition.
